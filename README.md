# refugee-conjoint-viz

Visualisation of results from a conjoint experiment in Turkey in 2018. For full description see 
Anna Getmansky, Konstantinos Matakos, Tolga Sinmazdemir, Diversity without Adversity? Ethnic Bias toward Refugees in a Co-Religious Society, International Studies Quarterly, Volume 68, Issue 2, June 2024, sqae031, https://doi.org/10.1093/isq/sqae031
